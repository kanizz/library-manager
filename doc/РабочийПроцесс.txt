Для того, чтобы скачать на ваш компьютер репозиторий делаем следующее:

===Скачивание репозитория===
* Убеждаемся что вы приняли приглашение от Камиля
* Открываем консоль
* Переходим на рабочий стол: cd Desktop
* Прописываем команду: git clone https://github.com/kanizz/library-manager
Вышеописанная команда скачает репозиторий и поместит его в папку library-manager

===Генерируем личный токен===
* Открываем GitHub. Жмём на правую верхнюю кнопку - кнопку профиля. Должно открыться боковое меню справа. Жмём на настройки (Settings)
* Жмём Developer Settings в левой панели
* Кликаем "Personal access tokens", затем "Tokens (classic)". Потом на "generate new token" и "generate new token (classic)"
* Заполняем поле Note, прокручиваем вниз, жмём "Generate token"
* СКОПИРУЙТЕ ПАРОЛЬ И СОХРАНИТЕ ЕГО ГДЕ НИБУДЬ

===Настройка репозитория===
* Заходим в скачанную папку: cd library-manager
* Пишем: git remote set-url origin git remote set-url origin https://ВАШ ТОКЕН@github.com/kanizz/library-manager.git

Если непонятно, следуйте инструкциям отсюда: https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens#creating-a-personal-access-token-classic или пишите в беседу

В папке doc лежит наша документация. Туда бросаем всё что пишем - диаграммы, модели, словари данных.
В папке src лежит выгрузка конфигурации. Как загрузить конфигурацию из этой папки - смотрите в видео в беседе.

Если вы сделали изменения в конфигурацию и хотите её загрузить в репозиторий, выгрузите конфигурацию в папку (см. видео в беседе) и затем напишите git push в консоли
